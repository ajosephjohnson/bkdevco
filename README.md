### Quick start

1.  **Build container and install application dependencies.**

    ```shell
    ./install.sh
    ```

2.  **Run the local development application.**

    ```shell
    docker-compose up -d
    ```

3.  **Start developing.**

    The application is now running at `http://localhost:3000`!
